static :修饰成员(成员变量,成员方法,内部类),进行数据共享(节省空间),构造方法默认静态,
static修饰的成员会被共享,将会被提升成为属于类的成员,(加载随类的加载而加载:例如静态代码块),将被存储在方法区内
且仅能调用类加载时存在的量或方法 不会被继承, 当子类有同名成员时将会被隐藏
abstract :修饰类 ,方法 ,接口 ,abstract修饰的方法无方法体,被继承后必需被override(重写) 
接口:只能有抽象方法public abstract ;只能有常量:public ( static ) final 数据类型 变量名.被implements时 必须public,接口中无构造(因此可以多实现:无需调用构造)
final :修饰的变量是常量,修饰的类不可以被继承,修饰的方法不可以被override
静态代码为静态变量进行初始化(当调用静态) 但调用.class时不会被调用(Object里没有)
接口中只能有常量即默认public static final 修饰的常量
                            和抽象方法:默认public abstract 修饰的方法

1.接口方法，访问权限必须是公共的 public
2.接口内只能有公共方法，不能存在成员变量,和成员常量
3.接口内只能含有未被实现的方法，也叫抽象方法，但是不需要写abstract关键字。
